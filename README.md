Aulas do YouTube DevDojo, Maratona Java, onde a disciplina do curso consiste em uma base sólida desde a configuração do ambiente de desenvolvimento (JDK e IDE) até o entendimento profundo de como a Java Virtual Machine (JVM) funciona. As etapas do aprendizado são:



Introdução



Operadores



Estruturas Condicionais



Estruturas de Repetição



Orientação a Objetos



Tratamento de Erros e Utilitários



Manipulação de Dados e Arquivos



Programação Moderna e Avançada
